# Требования к Telegram боту

## Основной функционал
- [x] Опросник с вопросами из JSON файла
- [x] Поддержка текстовых и вариантных вопросов
- [x] Сохранение ответов в CSV и JSON
- [x] Админ-панель (команда /admin)
- [x] Уведомления админу о новых ответах

## Версионирование
- [x] Хранение версий деплоя
- [x] Кнопка "Версия бота" в главном меню
- [x] Отображение текущей версии и даты деплоя

## Структура версий
```
versions/
├── v1.json  (дата, коммит, описание)
├── v2.json
└── v3.json
```

## Как добавить новую версию
1. Создайте файл `versions/v{N}.json` с описанием
2. Обновите `CURRENT_VERSION` в файле `config.py`
3. После деплоя бот автоматически покажет новую версию

## Формат файла версии
```json
{
  "version": "1.0.0",
  "date": "2026-02-09 12:00:00",
  "commit": "abc1234",
  "description": "Описание изменений"
}
```


## Какие файлы не надо изменять 
- Файл questions.json заполянется создателем, его не нужно менять
